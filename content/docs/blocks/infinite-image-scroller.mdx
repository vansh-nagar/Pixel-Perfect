## Infinite Image Scroller

Create beautiful infinite scrolling image galleries with smooth looping animations.

### Features

- Infinite loop animation
- Smooth scrolling
- Responsive grid
- Customizable speed

### Usage

```tsx
import InfiniteImageScroller from "@/pixels/infinite-image-scroller";

export default function Page() {
  return <InfiniteImageScroller />;
}
```

### Props

- `images` - Array of image URLs
- `direction` - Scroll direction ("left" | "right")
- `speed` - Animation speed (default: 1)
