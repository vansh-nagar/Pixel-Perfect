## Text Fade

Create beautiful text animations with smooth fade effects for engaging typography.

### Features

- Smooth fade animations
- Customizable speed
- Word-by-word effects
- Responsive text sizing

### Usage

```tsx
import TextFade from "@/pixels/text-fade";

export default function Page() {
  return <TextFade />;
}
```

### Options

- `text` - Text content
- `speed` - Animation speed
- `delay` - Stagger delay between words
