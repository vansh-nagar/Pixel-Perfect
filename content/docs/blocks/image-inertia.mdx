## Image Inertia

Display images with smooth inertial scrolling animations for an engaging user experience.

### Features

- Inertial scroll effects
- Smooth image transitions
- Performance optimized
- Responsive images

### Usage

```tsx
import ImageInertia from "@/pixels/image-inertia";

export default function Page() {
  return <ImageInertia />;
}
```

### Props

- `src` - Image source URL
- `alt` - Alternative text
- `className` - Custom styling
