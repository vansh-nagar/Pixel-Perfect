## Ring

Display animated rings with rotating effects and customizable styling options.

### Features

- Smooth rotation animations
- Customizable colors
- Responsive design
- Multiple ring styles

### Usage

```tsx
import Ring from "@/pixels/ring";

export default function Page() {
  return <Ring />;
}
```

### Props

- `size` - Ring diameter
- `color` - Ring color
- `speed` - Rotation speed
