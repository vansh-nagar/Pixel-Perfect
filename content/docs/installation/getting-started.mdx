## Installation

### Install Next.js

Get started with Next.js by installing it in your project:

```bash
npm install next react react-dom
```

### Install Tailwind CSS

Set up Tailwind CSS for styling:

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Add utilities

Add Tailwind utility classes to your project configuration:

```ts
// tailwind.config.ts
export default {
  content: [
    "./pages/**/*.{ts,tsx}",
    "./components/**/*.{ts,tsx}",
    "./app/**/*.{ts,tsx}",
  ],
};
```

### CLI

<span className="inline-flex items-center gap-2">**3.0**</span>

Use our command-line interface to manage components:

```bash
npm install -g pixel-perfect-cli
pixel-perfect-cli init
pixel-perfect-cli add button
```
