## Mouse Follower 2

An enhanced version of the mouse follower with additional visual effects and customization options.

### Features

- Advanced cursor tracking
- Multiple trailing effects
- Customizable particle effects
- Enhanced animations

### Usage

```tsx
import MouseFollower2 from "@/pixels/mouse-follower-2";

export default function Page() {
  return <MouseFollower2 />;
}
```

### Customization

Adjust the animation behavior and visual style:

```tsx
<MouseFollower2 trailLength={10} particleCount={5} color="gradient" />
```
