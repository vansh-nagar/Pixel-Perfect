## Text Inertia

Display text with inertial scrolling animations for smooth, interactive typography effects.

### Features

- Inertial scroll effects
- Smooth text transitions
- Interactive animations
- Performance optimized

```tsx
import TextInertia from "@/pixels/text-inertia";

export default function Page() {
  return <TextInertia />;
}
```

### Configuration

```tsx
<TextInertia text="Your text here" friction={0.9} velocity={1} />
```
