## Cards Stack

Create beautiful stacked card animations with smooth transitions and gesture support.

### Features

- Smooth card stacking animation
- Swipe gesture support
- Responsive layout
- Customizable styling

### Usage

```tsx
import CardsStack from "@/pixels/cards-stack";

export default function Page() {
  return <CardsStack />;
}
```

### Customization

Modify the card content and styling to match your design:

```tsx
<CardsStack
  cards={[
    { title: "Card 1", content: "..." },
    { title: "Card 2", content: "..." },
  ]}
/>
```
