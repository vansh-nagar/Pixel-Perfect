## Install Next.js

Get started by creating a new Next.js project with Create Next App.

### Create a new project

Run the following command to create a new Next.js application:

```bash
npx create-next-app@latest
```

### Configuration prompts

During installation, you'll be presented with the following prompts:

```
? What is your project named? my-app
? Would you like to use TypeScript? No / Yes
? Would you like to use ESLint? No / Yes
? Would you like to use Tailwind CSS? No / Yes
? Would you like to use `src/` directory? No / Yes
? Would you like to use App Router? (recommended) No / Yes
? Would you like to customize the default import alias (@/*)? No / Yes
? What import alias would you like configured? @/*
```

### Recommended settings

For this project, we recommend:

- **TypeScript**: Yes - Adds type safety to your project
- **ESLint**: Yes - Helps maintain code quality
- **Tailwind CSS**: Yes - Required for styling
- **src/ directory**: Yes - Better project organization
- **App Router**: Yes - Modern Next.js routing
- **Import alias**: Yes - Use `@/*` for cleaner imports

### Start the development server

Navigate to your project directory and start the development server:

```bash
cd my-app
npm run dev
```

Your application will be available at `http://localhost:3000`.
